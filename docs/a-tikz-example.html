Example of TikZ figure generation with the Graph plugin

The Graph plugin collect the interface to some useful image generation programs. Whenever possible a vector image in the PostScript or Pdf format is generated. Currently it supports the Graphviz suite and some LaTeX image generation packages like XyPic, FeynMF, PGF/TikZ and a direct interface to LaTeX/PDFLaTeX for more complex layout or to use other packages.

You may insert a Graph session using Insert→Session→Graph. When you finished typing your input, press Return in order to compute the output. Multiline input is entered using Shift+Return.

Type help, you will get all the available graph backends.

\documentclass[tikz]{standalone}

\usepackage{tikz-3dplot}

\begin{document}

\tdplotsetmaincoords{60}{120}

\begin{tikzpicture}[scale=3,tdplot_main_coords,axis/.style={->,blue,thick},

vector/.style={-stealth,red,very thick},vector guide/.style={dashed,red,thick}]

%standard tikz coordinate definition using x, y, z coords

\coordinate (O) at (0,0,0);

%tikz-3dplot coordinate definition using r, theta, phi coords

\tdplotsetcoord{P}{.8}{55}{60}

%draw axes

\draw[axis] (0,0,0) – (1,0,0) node[anchor=north east]{$x$};

\draw[axis] (0,0,0) – (0,1,0) node[anchor=north west]{$y$};

\draw[axis] (0,0,0) – (0,0,1) node[anchor=south]{$z$};

%draw a vector from O to P

\draw[vector] (O) – (P);

%draw guide lines to components

\draw[vector guide] (O) – (Pxy);

\draw[vector guide] (Pxy) – (P);

\end{tikzpicture}

\end{document}
